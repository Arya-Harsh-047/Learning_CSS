CSS 2D Transforms


  • CSS transforms allow you to move, rotate, scale, and skew elements.



    • CSS 2D Transforms Methods:



    1. The translate() Method: 

        
    
    The translate() method moves an element from its current position (according to the parameters given for the X-axis and the Y-axis). 
    
    Example moves the div element 50 pixels to the right, and 100 pixels down from its current position.
    Hover over me to see effect of translate 50px on X-axis & 100px on Y-axis.

    2. The rotate() Method:

        
    
     The rotate() method rotates an element clockwise or counter-clockwise according to a given degree. 

     Example:  rotates the div element clockwise with 20 degrees.
    Hover over me to see effect of rotate clockwise 20 degree.

    3. The scale() Method:

        
    
    The scale() method increases or decreases the size of an element (according to the parameters given for the width and height).

    Example: increases the div element to be 1.5 times of its original width, and 1.5 times of its original height:
    Hover over me to see effect of scaling by 1.5 times original width size & 1.5 times of its original height.

    4. The skew() Method: 

        
    
    The skew() method skews(तिरछा) an element along the X and Y-axis by the given angles.

    Example: skews the div element 20 degrees along the X-axis, and 10 degrees along the Y-axis:
    Hover over me to see effect of skew 20 degrees along the X-axis, and 10 degrees along the Y-axis.

    5. The matrix() Method: 

        
    
    The matrix() method combines all the 2D transform methods into one.

    The parameters are as follow: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY())

    Example:   transform: matrix(1, -0.3, 0, 1, 0, 0);
    Hover over me to see effect of matrix transformation.

    6. Transform Origin: 

          
    
        The origin from which a transformation takes place is known as the "transform-origin". It can be set using CSS property 'transform-origin'.
    Hover over me to see effect of origin (center).
