CSS Padding

    Padding is used to create space around an element's content, inside of any defined borders.


    This element has a padding of 30px



Padding - Individual Sides

 CSS has properties for specifying the Padding for each side of an element:

   1. Padding-top
   2. Padding-right
   3. Padding-bottom
   4. Padding-left

All the padding properties can have the following values:
    
   a. auto - the browser calculates the Padding
   b. length - specifies a Padding in px, pt, cm, etc.
   c. % - specifies a Padding in % of the width of the containing element
   d. inherit - specifies that the Padding should be inherited from the parent element

Tip: Negative values are allowed.

    Example:

Set different Paddings for all four sides of this "p" element:




If the Padding property has three values:

    
 Padding: 25px 50px 45px;

Here,
    top Padding is 25px
    right and left Paddings are 50px
    bottom Padding is 45px


If the Padding property has two values:

    
 Padding: 25px 50px;

Here,
    top and bottom Paddings are 25px
    right and left Paddings are 50px


If the Padding property has one value:

    
 Padding: 25px 

Here,
    all four Paddings are 25px
    all four Paddings are 25px
